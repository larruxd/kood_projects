# Pacman


## Description
Its Pacman.

## How To Run
1. Clone the repo.
2. Use ```go run server.go``` in the terminal and go to [http://localhost:8080] in your browser.

## How to Play
1. Press start.
2. Arrow keys to move.
3. "Esc" or "Space" keys to pause/unpause the game.
